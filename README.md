# Bloom — Expense Tracker

Gentle expense tracker + daily scheduler. Vite + React + TS. Dark/light, EN/DE, localStorage.

## Run

```bash
npm install
npm run dev
```

Build: `npm run build`

## Features

- Expenses: add, categorize, delete, monthly total
- Schedule: plans with date/time, check off when done
- Summary: totals + category bars
- Theme toggle (light/dark), language toggle (EN/DE)
- All data persisted in localStorage
